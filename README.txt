Extract the folder

To run file click on kode.cmd
	or
Open cmd at the folder "Kode Incubating", type kode and press enter.




Pre-requirment ->

JDK must be present and JAVA_HOME is set correctly.


Supported Tokens ->
	Numbers, +, -, *, /, (, ), and ;

	Example -> 1+(2*3);

To exit -> type "exit" into the shell and press enter.
